# Code of conduct

We are committed to a respectful, welcoming, and harassment-free project for
everyone. Be constructive, assume good intent, respect differing viewpoints,
and focus criticism on ideas and code rather than people.

Unacceptable behavior includes harassment, discrimination, threats, deliberate
intimidation, publishing private information, or sustained disruption. Project
maintainers may remove content or restrict participation when necessary to keep
the community safe.

Report conduct concerns privately to the Usermaven maintainers through the
contact options on the Usermaven GitHub organization.
